# EZDashboard builder

把 Excel / CSV 一鍵變成精美、可互動的網頁儀表板產生器。

- **上傳即用**：拖放 Excel / CSV，自動分析欄位型別並產生 KPI 指標卡與圖表。
- **髒表格整理**：上傳時可選標題列、略過雜訊列、自動偵測標題、交叉表反轉 (Unpivot)、CSV 編碼切換 (UTF-8 / Big5)。
- **自訂運算**：兩欄快速運算或自由公式（`[銷售額] * [數量] - [成本]`，支援括號與優先序）。
- **版面與主題**：範本/自訂卡片、篩選器、淺/深色模式、主題配色（含自訂色）。
- **一鍵下載**：匯出成單一檔案、資料完全內嵌、可離線雙擊開啟的互動式 HTML 儀表板。

不需任何 API 金鑰或後端服務，純前端 SPA。

## 本機開發

**需求：** Node.js

1. 安裝相依套件：`npm install`
2. 啟動：`npm run dev`
3. 開啟 http://localhost:3000

## 建置 / 部署

- `npm run build` — 產生靜態網站到 `dist/`（可直接部署到 Netlify 等靜態主機）。
- `npm run build:single` — 產生**單一檔案** `dist/index.html`，可直接雙擊離線使用、或分享給他人。
